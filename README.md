# Sibers Project Frontend

Фронтенд-приложение для управления проектами и сотрудниками.

## Функционал

- Вкладка **Проекты**:
  - просмотр списка проектов;
  - фильтрация проектов;
  - создание проекта;
  - просмотр и редактирование проекта;
  - управление участниками проекта;
  - загрузка/удаление документов проекта.

- Вкладка **Сотрудники**:
  - просмотр списка сотрудников;
  - создание сотрудника;
  - просмотр карточки сотрудника;
  - редактирование сотрудника;
  - удаление сотрудника.

## Технологии

- Vue 3 + TypeScript
- Pinia
- Vue Router
- Axios

## Запуск проекта

1. Установить зависимости:

```bash
npm install
```

2. Запустить dev-сервер:

```bash
npm run dev
```

3. Открыть приложение в браузере по адресу из вывода Vite (обычно `http://localhost:5173`).

## Проверка качества кода

```bash
npm run lint
```
